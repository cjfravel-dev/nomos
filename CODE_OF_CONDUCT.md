# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment and abusive behavior are not tolerated.

To report unacceptable behavior, contact the maintainer at **cj@cjfravel.dev**. Reports will be
handled confidentially.

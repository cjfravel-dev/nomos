JsonNumber 

Companion object JsonNumber

final case class JsonNumber(raw: String) extends JsonValue with Product with Serializable

A JSON number, kept as its original textual lexeme so output round-trips exactly (e.g. 30.0 stays 30.0). Typed accessors derive values lazily and exactly.

raw

the original (or canonical) number text; validated on construction to be a well-formed JSON number lexeme so the writer can emit it verbatim as valid JSON

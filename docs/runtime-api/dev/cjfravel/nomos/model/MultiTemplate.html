MultiTemplate 

Companion object MultiTemplate

case class MultiTemplate(basePackage: String, definitions: List[TemplateDefinition], listType: String = "List", fromJsonStyle: String = "either", dateType: String = "java.time.LocalDate", dateTimeType: String = "java.time.LocalDateTime", mapType: String = "Map", visibility: Option[String] = None) extends Product with Serializable

Represents a complete template file with multiple type definitions.

basePackage

Base package for all generated code (derived from template path)

definitions

List of type definitions

listType

The collection type to use for arrays: "List" or "Array" (default: "List")

mapType

The collection type to use for $$map fields: "Map" or "java.util.Map" (default: "Map")

visibility

Optional access modifier (e.g. "private[incentives]") prepended to every generated top-level definition; absent means public

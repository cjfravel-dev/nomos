Overview

Nomos generates Scala model code from a JSON template. Previously, validating JSON against that template required the original template file at runtime. Template embedding serializes the entire MultiTemplate into the generated NomosFormats object, so every generated model can validate JSON on its own — no template file, no extra wiring. Ideal for production where templates aren't shipped.

How it works

nomos/templates/com/example/models/user.json→ CodeGenerator→ TemplateSerializer→ NomosFormats.embeddedTemplate→ MultiValidator→ User.validate(json)

At generation time, TemplateSerializer turns the parsed MultiTemplate into Scala source. That source is written into NomosFormats as a lazy val, and a MultiValidator is built from it. Each model companion gains a validate() that runs the validator, then parses.

Layout-driven config: templates live under src/main/resources/nomos/templates/<package path>/. The base package is derived from that path (no basePackage, outputDir, or mainClass fields), and validate() targets each model by its fully-qualified name, e.g. "com.example.models.user.User".

NomosFormats (excerpt, real output)

object NomosFormats {
  // Embedded template for runtime validation
  lazy val embeddedTemplate: MultiTemplate = {
    MultiTemplate(
      basePackage = "com.example.models",
      definitions = List(TemplateDefinition(
        name = "User",
        templateType = ObjectType(ListMap(
          "id" -> FieldDef(StringType(List()), optional = false),
          "age" -> FieldDef(NumberType(List()), optional = true),
          "roles" -> FieldDef(ArrayType(StringType(List())), optional = false))),
        subPackage = Some("user"))),
      useOptionTypes = true, listType = "List")
  }
  lazy val validator: MultiValidator = new MultiValidator(embeddedTemplate)
}

Per-model validate() method

def validate(json: String): Either[List[ValidationError], User] = {
  validator.validate(json, "com.example.models.user.User") match {
    case Right(_)      => fromJson(json).left.map(e => List(ValidationError("root", e, "valid JSON", json)))
    case Left(errors) => Left(errors)
  }
}

Generated for all three model shapes: case classes, sealed-trait variant groups, and type-discriminator unions.

Usage

import com.example.models.user.User

User.validate("""{"id":"1","username":"bob","email":"b@x.com","roles":["a"]}""")
// => Right(User(...))

User.validate("""{"username":"bob","email":"b@x.com"}""")
// => Left(List(ValidationError(id, ...), ValidationError(roles, ...)))
